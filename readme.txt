Contact Form 7 Addons Plugin

== Description ==
Formxtra CF7 is a plugin for extending Contact Form 7 plugin functionality.

== Installation ==
1. Upload the `formxtra-cf7` folder to the `/wp-content/plugins/` directory or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
